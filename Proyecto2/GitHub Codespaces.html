GITHUB Codespaces


Un «Codespace» es un entorno de desarrollo integrado incluido en GitHub y disponible para ser usado en cualquier repositorio. Al igual que otros entornos, el entorno de GitHub le permite al desarrollador editar, depurar, ejecutar y realizar acciones del sistema de versiones directamente desde GitHub, sin la necesidad de clonar el repositorio en la computadora.Es posible personalizar los Codespaces al gusto: cambiar el tipo de máquina, añadir un contenedor de desarrollo, administrar los secretos cifrados, etc. Con relación a esto, GitHub ofrece una versión similar de forma gratuita (naturalmente con menos ventajas y posibilidades) que ayuda a los desarrolladores a interactuar con los repositorios que quieran sin la necesidad de abrir un editor de texto.
